# Warudo 풍벽지 (BalloonWall)

**Warudo 풍벽지**는 방송 도네이션을 시각적으로 표현하는 인터랙티브 오버레이 앱입니다.
도네이션이 카드 형태로 떨어지고, 쌓이고, 드래그로 정리할 수 있는 풍선벽을 만들어보세요.

## 주요 기능

- **카드 스태킹** — 도네이션이 카드 형태로 자동 스택
- **드래그 앤 드롭** — 마우스로 카드/스택을 자유롭게 이동 및 정리
- **자동 정렬** — 흩어진 카드를 깔끔하게 자동 배열
- **티어별 디자인** — 금액별 카드 이미지 및 색상 커스터마이징 (Bronze → Gold)
- **추가시그 (커스텀 풍선)** — 원하는 이미지로 나만의 풍선 카드 생성
- **OBS 연동** — 브라우저 소스로 직접 삽입 가능
- **WebSocket API** — 외부 프로그램에서 도네이션 이벤트 전송
- **세션 관리** — 레이아웃 저장 및 불러오기
- **자동 업데이트** — 새 버전 출시 시 앱 내 자동 업데이트 알림
- **피드백 시스템** — 앱 내에서 개발자에게 직접 의견 전달

## 설치 방법

1. [Releases](https://github.com/dinoosaur726/BalloonWall/releases) 페이지에서 최신 `BalloonWall-Setup-x.x.x.exe`를 다운로드합니다.
2. 설치 파일을 실행하고 안내에 따라 설치합니다.
3. 앱을 실행하면 바로 사용할 수 있습니다.

업데이트가 있으면 앱 실행 시 자동으로 알림이 표시됩니다.

## 사용 방법

### 기본 조작

| 동작 | 설명 |
|------|------|
| 드래그 | 카드/스택 이동 |
| 스크롤 | 호버된 스택 크기 조절 |
| ESC | 설정 창 열기/닫기 |

### 설정

앱 실행 후 `ESC` 키를 눌러 설정 창을 열 수 있습니다. 설정에서 다음 항목을 조정할 수 있습니다:

- **WebSocket 포트** — 도네이션 수신 포트 (기본: 3005)
- **HTTP 포트** — OBS 브라우저 소스 포트 (기본: 3006)
- **스트리머 ID** — 시그니처 풍선 이미지 연동
- **자동 추가** — 도네이션 수신 시 자동으로 카드 생성
- **최소 금액** — 설정 금액 이상의 도네이션만 카드로 생성

### OBS 브라우저 소스 연동

OBS에서 브라우저 소스를 추가하고 다음 URL을 입력합니다:

```
http://localhost:3006
```

포트 번호는 설정에서 변경할 수 있습니다.

### WebSocket 연동

외부 프로그램에서 WebSocket으로 도네이션 이벤트를 전송할 수 있습니다.

- **URL**: `ws://localhost:3005` (포트는 설정에서 변경 가능)
- **메시지 형식**: `풍선타입/닉네임/금액` 또는 `닉네임/금액`

```javascript
const ws = new WebSocket('ws://localhost:3005');

// 일반 풍선
ws.onopen = () => ws.send('Normal/팬123/5000');

// 광고 풍선
ws.onopen = () => ws.send('Ad/팬123/5000');

// 타입 생략 시 일반 풍선으로 처리
ws.onopen = () => ws.send('팬123/5000');
```

### 추가시그 (커스텀 풍선)

설정 창의 "추가시그" 탭에서 원하는 이미지와 금액을 등록하면, 해당 금액의 도네이션이 들어올 때 커스텀 이미지가 표시됩니다.

- 일반 풍선, 광고 풍선 각각 적용 여부를 선택할 수 있습니다.
- 권장 이미지 사이즈: 480 x 285 px (비율 16:9.5)
- 빈 여백 없이 꽉 찬 이미지를 사용하면 카드에 깔끔하게 표시됩니다.
